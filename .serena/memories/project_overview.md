Yakyuken Kingdom is a Godot 4-based 3D rock-paper-scissors (card-battle style) game. The runtime project lives entirely under `godot/` (open `godot/project.godot` with Godot 4.2+). Gameplay scenes include `Main.tscn` for the core flow and `Table3D.tscn` for the full 3D battle view; HUD-specific debugging can be done with `TestDisplay.tscn`. Supporting docs/specs live at the repo root (`game_spec.md`, etc.), while reusable art prompts/reference materials stay in `assets/`. Godot assets/resources/timelines need to be kept in sync when edited to preserve diffability.
