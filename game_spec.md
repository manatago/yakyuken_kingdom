# じゃんけんゲームの概要
- じゃんけんゲームはトランプのようなカード（グー、チョキ、パーの3種類）を使用する
- 対戦は、通常は1対1とする。将来の拡張で多人数でも可能なようにする
- 手持ち札は、6枚とするが、こちらも簡単に変更できるようにする（手札は固定で、全種類2枚）
- じゃんけんに勝つと、負けた方が買った方にコインを渡す。
- コインの最初の手持ちは3枚とする。コインが全てなくなると負けになる。
- 最期までいった場合は、手持ちのコインが多い方が勝ちとなる。

# 画面
- デッキと手持ちの2種類があり、手持ちは自分の未使用のカード。デッキは両者が勝負済のカード
- 相手の上半身が見えるようする
とりあえず、静止画でOK

MVPはブラウザで動作するようにする。製品版になった時にRustのTauriで実装する


# 修正指示(No1)
じゃんけんの瞬間が表示されなくて、文字だけなのでとても味気ない。
- 相手のカードと自分のカードを表向けて、勝負が見えるようすること。少しビジュアルをリッチにする。
- 過去の札は伏せない。表を向けること。


# 修正指示(No2)
- 手持ちはコインではなく、服の数とする。服は脱ぐだけなので、勝手も自分の服は増えない-。
- 勝負の画像は、左右になっているが、上下にする。
- 同時に表示させるのではなく、相手の分はあとから表示刷るようにする。
- 勝ち負けの表示をだすのも間を持たせる
