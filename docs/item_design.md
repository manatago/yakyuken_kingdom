# アイテム設計

じゃんけんキングダムのアイテム仕様書。仕様が確定したものから実装する。

## 共通事項

### 種別
- **消耗品（Consumable）**: 使い切りでインベントリから消費される
- **装備品（Equipment）**: バトル中は常時発動、装備枠を占有

### 入手経路（暫定）
- バトル報酬（ランダムドロップ）
- ショップ購入（未実装）
- イベント報酬（シナリオで付与）

### 使用タイミング
- バトル前: デッキ構築画面でアイテム選択
- バトル中: じゃんけん発動前にアイテムボタンで使用
- 常時発動: 装備品（バトル開始時自動で効果適用）
