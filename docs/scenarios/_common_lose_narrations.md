# サトシ敗北時の共通ナレーション・プール

各サブイベント／本編ステージでサトシがバトルに敗北した際、
個別の脱衣描写を毎回書き起こす代わりに、本ファイルに登録された
パターンからランダムに 1 つ選んで表示する。

## 設計方針

- **目的**: 章ごとの脱衣描写を毎回書き直す手間を削減し、トーンを統一する。
- **位置づけ**: 各章固有の「敗北状況」ナレ (1〜2 行) の **直後** に挿入。
    例: シスター長戦敗北 →
        「番兵に拘束された…」 (章固有)
        + 共通ナレーション (ランダム 1 件)
        → ギルドホーム送還
- **描写レベル**: 「脱がされ始め」までで打ち止め。下着姿は描かない。
    上着のボタンが弾ける／ベルトが緩む／シャツが捲れる
    といった示唆的演出に留め、ピー助が「撤退」を促して打ち切る。
- **ランダム抽出**: 1 セッション中、直前に出たパターンは可能なら
    回避する（実装時に重複回避の責務を持つ）。

## 実装規約

- 各バトルチャプターの `get_lose_redirect()` から呼ばれる
  story_sequence_then_guild_home 系のシーケンスは、章固有のナレ後に
  以下のいずれか 1 件を表示してからギルドホームに戻る。
- 共通ナレーションを表示するための DSL コマンド
  `b.satoshi_lose_narration()` を新設予定。
- パターン定義は実装側 (`BattleChapterBase.SATOSHI_LOSE_NARRATIONS`)
  に const として保持し、本ドキュメントの内容と同期させる。
- パターンを追加・修正したら本ドキュメントとコードの両方を更新する。

## 設定上の前提

### 敗北時の脱衣メカニクス

カードバトルに敗北すると、装着しているチップが装着者の体を一時的に
制御し、**自分の手で衣服を脱がせる**。脱ぎ終わるとチップの制御は
解除され、本人の意思に戻る。

- 相手が脱がすのではなく、本人が（チップ制御で）自ら脱ぐ
- 脱いだ衣服は床に散らばる
- そこから先（衣服を奪い返す／土下座で返してもらう／取られる など）
  はパターン分岐

### キャラ前提

- **サトシ**: 小心者・ムッツリスケベ・情けない男。
  泣き崩れる／土下座する／半泣きで逃げる など、徹底して情けない挙動。
  ただし **マザコンではないので「お母さん」とは叫ばない**。
- **ピー助**: サトシの**対等な相棒**。主人と従者の関係ではない。
  口調は「お前」「サトシ」と砕けた呼び方、呆れたツッコミ、
  そして必要時には引っ張ってでも逃がす相棒キャラ。
  ナレ地の文でも「主人」とは決して書かない（「サトシ」と書く）。

### 相手プレースホルダー

各パターン内の `{opponent}` は実行時に
`BattleChapterBase.get_opponent_name()` の値で置換される。
プレースホルダーは **地の文の主語** にのみ使用し、
**直接対話 (`{opponent}:` セリフ) は使わない**
（キャラごとの口調差を吸収できないため）。

## ナレーション・パターン (12 種、A/B/C/D 各 3 バリアント)

サトシは小心者・ムッツリスケベ・情けない男の典型。
全パターンとも「カッコ悪く / 情けなく / 取り乱して」逃げるトーンで統一する。

### A. 奪い返して泣き逃げ (服回収・カード喪失のみ)

#### A-1. 制御解除と同時に強奪

```
チップが発動し、サトシの両手が勝手に動き出す。
意思に反して、襟元を緩め、ボタンを外し、シャツを脱ぎ捨てる。
続いてベルトが緩み、ズボンが床に滑り落ちた。
制御が外れた瞬間、サトシは床に散らばった衣服に飛びつき、
{opponent} の手が伸びるより先にひったくった。
サトシ:
  ご、ごめんなさいごめんなさい！
半裸で衣服を抱え、サトシは脱兎のごとく駆け出した。
```

#### A-2. 半泣きで床に転がる

```
チップに体を乗っ取られ、サトシは無表情のままシャツを脱ぎ、
ズボンを下ろし、床にきれいに揃えて置いた。
制御が解けると同時にサトシは床に転がり、
散らばった服を必死にかき集めた。
サトシ:
  俺の、俺のだから……！
両手で衣服を抱え込み、半裸のまま、
サトシは情けない悲鳴を上げて駆け出した。
```

#### A-3. 死に物狂いの捕獲

```
チップが発動。意思を奪われたサトシは、
シャツのボタンを一つずつ外し、ズボンのベルトを緩め、
衣服を一枚ずつ床に脱ぎ落としていく。
制御が外れた瞬間、サトシは目を血走らせて床の服に飛びついた。
サトシ:
  う、うわぁぁ、返せ、返せぇぇ！
涙と鼻水でぐしゃぐしゃになりながら、
サトシは服を抱え、転げるように退散した。
```

### B. 土下座号泣で返してもらう (服回収・カード喪失のみ)

#### B-1. 全力土下座

```
チップが発動。サトシの体が勝手に動き、シャツが、ズボンが、
ひとつずつ床に脱ぎ落とされていく。
意思を取り戻したサトシは、両手で前を押さえてその場で土下座した。
サトシ:
  ご、ごめんなさいごめんなさいごめんなさいぃぃ！
  ふ、服だけは、服だけは返してくださいぃぃ！
涙と鼻水で床を濡らし、何度も何度も額を擦りつけて詫びた。
{opponent} は呆れた様子で、足元の衣服をぽいと放り投げた。
ピー助:（小声）
  ...プライドってもんが無いのか、お前は。
サトシは服を抱きしめ、半泣きで退散した。
```

#### B-2. 命乞いセット

```
チップに乗っ取られたサトシは、自らの手でシャツを脱ぎ、
ベルトを緩め、ズボンを床に落とした。
制御が解けると同時に、がくがくと膝をつき、
頭を床に擦りつけた。
サトシ:
  な、なんでもします、なんでもしますからぁぁ！
  ふ、服だけは、お情けを、お情けをぉぉ！
涙でぐしゃぐしゃになった顔で詫び続けるサトシに、
{opponent} はため息混じりに服を投げて寄越した。
サトシ:
  あ、ありがとうございますぅぅ！
情けなく感謝を述べ、サトシは衣服を抱えて駆け出した。
```

#### B-3. 裸土下座

```
チップが発動。サトシは無表情のまま、ほぼ全ての衣服を
ひとつずつ床に脱ぎ落としていった。
制御が解けたサトシは、両手で前を押さえ、その場で土下座した。
サトシ:
  許してくださいぃ、ふ、服を、服をどうかぁぁ！
小刻みに震える肩、涙で曇る視界、額を必死に床に擦り続ける。
{opponent} は呆れ果てて、衣服を「持ってけ」と放り投げた。
サトシは抱きしめるように服を取り、よろよろと退散した。
```

### C. ピー助が言いくるめて返してもらう (服回収・カード喪失のみ)

#### C-1. 煽て倒し

```
チップが発動。サトシは意思に反してシャツのボタンを外し、
ベルトを緩め、ズボンを床に落とした。
為す術もなく両手で前を押さえるサトシの代わりに、
ピー助が囁き始めた。
ピー助:（サトシの声で）
  ...いやぁ、見事な技でした。
  あなたほどの方なら、敗者の服など興味ないでしょう？
  あれは家族の形見でして、私の手元にあって初めて意味があるもの。
{opponent} は満更でもない顔で、足元の衣服をぽいと投げて寄越した。
ピー助:（小声）
  ...サトシ、いいか、いつ何時もプライドはくすぐっておけ。
サトシは情けない顔のまま、衣服を抱えて退散した。
```

#### C-2. 脅し気味の交渉

```
チップに体を奪われ、サトシはシャツとズボンを次々に脱ぎ捨てた。
口をパクパクさせて言葉が出ないサトシの代わりに、
ピー助が囁いた。
ピー助:（サトシの声で）
  ...これ、王都中央広場で晒されたら、あなたの方も
  品位を疑われますよ。
  「冒険者を全裸にして喜ぶ」と。
  ...返していただけますね？
{opponent} は鼻白んで、衣服を投げ寄越した。
ピー助:（小声）
  ...口は災いの元、ってのを利用するんだよ。
サトシは涙目でうなずきながら、服を掻き集めて駆け出した。
```

#### C-3. 同情を誘う作り話

```
チップ発動でシャツが脱げ、ズボンが床に落ちる。
サトシは半泣きで何も言えない。
ピー助:（サトシの声で・しおらしく）
  ...すみません。こいつ、本当にこれしか着るものがないんです。
  実家から仕送りも止まってまして...。
  どうか、ご慈悲を...。
{opponent} は呆れた様子で、「持ってけ、馬鹿が」と
衣服を投げて寄越した。
ピー助:（小声）
  ...嘘も方便ってな。お前、感謝しろよ。
サトシは服を抱え、ぐすぐすと泣きながら退散した。
```

### D. 服を取られて手ぶらで逃げる (服 ✗・カード喪失 + 所持金 -X)

#### D-1. 安値で売り捌かれる

```
チップが発動。サトシは自分の手で、シャツも、ズボンも、
一枚ずつ床に脱ぎ落とした。
{opponent} は床の衣服を見下ろし、つまむように拾い上げた。
汚れた男の使い古しなど、誰も家宝にはしない。
だが裏路地の古着屋に流せば、銅貨数枚にはなる。
両手で前を押さえ、サトシはがたがた震える。
ピー助:（焦った声で）
  サトシ、もう諦めろ！ 走れ、走るんだ！
サトシ:
  う、うわぁぁぁ、嫌だ嫌だ嫌だぁぁぁ！
人目を避ける場所を必死に探しながら、
サトシは半裸でギルドへ駆け戻った。
（後日、新しい服を購入：所持金 -X）
```

#### D-2. 完全黙殺

```
チップに体を奪われ、サトシは無感情にシャツとズボンを脱ぎ、
床に揃えて置いた。
{opponent} はその衣服を一瞥もせず、足で蹴り散らかしてから
部屋を出ていった。
サトシは床に散った服に手を伸ばすが、ピー助が叫んだ。
ピー助:
  サトシ、それ、もう拾うな！ 罠かもしれん！
  ...今は、逃げろ！
サトシ:
  で、でも俺の服がぁぁ...！
ピー助に強引に引かれ、サトシは半裸で人目を忍びつつ、
ギルドへ逃げ帰った。
（後日、新しい服を購入：所持金 -X）
```

#### D-3. ぞんざいに古着屋行き

```
チップが発動。サトシは意思とは無関係に、シャツとズボンを
ひとつずつ脱ぎ捨てた。
{opponent} は床の衣服を、ぞんざいに拾い上げた。
古着屋に持ち込めば、捨てるよりはマシな値が付く。
それすら惜しい代物だが、「一応」貰っておく、という顔で。
サトシ:
  そ、そんなぁぁ！
両手で前を押さえ、その場でへたり込むサトシ。
ピー助:（深いため息）
  ...諦めろ、サトシ。買い直すぞ。
ピー助はサトシの腕を引き、半裸で号泣するサトシを、
裏路地伝いにギルドまで連れ帰った。
（後日、新しい服を購入：所持金 -X）
```

## 公開・中継系の例外条項

サブイベント 3 のように「真言の水晶球＋伝声塔」など、サトシの姿が
公衆に中継されうる演出が組み込まれている場合は、サトシ敗北時は
**必ず中継が停止／切断／失敗する** 設定にする。サトシの姿は世間に
晒されない。これは本ドキュメントのナレーション・パターンに優先する
ハードルールとして守る。
