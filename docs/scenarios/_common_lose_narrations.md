# サトシ敗北時の共通ナレーション・プール

各サブイベント／本編ステージでサトシがバトルに敗北した際、
個別の脱衣描写を毎回書き起こす代わりに、本ファイルに登録された
パターンからランダムに 1 つ選んで表示する。

## 設計方針

- **目的**: 章ごとの脱衣描写を毎回書き直す手間を削減し、トーンを統一する。
- **位置づけ**: 各章固有の「敗北状況」ナレ (1〜2 行) の **直後** に挿入。
    例: シスター長戦敗北 →
        「番兵に拘束された…」 (章固有)
        + 共通ナレーション (ランダム 1 件)
        → ギルドホーム送還
- **描写レベル**: 「脱がされ始め」までで打ち止め。下着姿は描かない。
    上着のボタンが弾ける／ベルトが緩む／シャツが捲れる
    といった示唆的演出に留め、ピー助が「撤退」を促して打ち切る。
- **ランダム抽出**: 1 セッション中、直前に出たパターンは可能なら
    回避する（実装時に重複回避の責務を持つ）。

## 実装規約

- 各バトルチャプターの `get_lose_redirect()` から呼ばれる
  story_sequence_then_guild_home 系のシーケンスは、章固有のナレ後に
  以下のいずれか 1 件を表示してからギルドホームに戻る。
- 共通ナレーションを表示するための DSL コマンド
  `b.satoshi_lose_narration()` を新設予定。
- パターン定義は実装側 (`BattleChapterBase.SATOSHI_LOSE_NARRATIONS`)
  に const として保持し、本ドキュメントの内容と同期させる。
- パターンを追加・修正したら本ドキュメントとコードの両方を更新する。

## ナレーション・パターン (5 種)

サトシは小心者・ムッツリスケベ・情けない男の典型。
全パターンとも「カッコ悪く / 情けなく / 取り乱して」逃げるトーンで統一する。

### A. 半泣き敗走 (滑稽・テンポ良い)

```
ボタンが次々と弾け飛ぶ。「今度こそ無理だ！」
サトシは半泣きで衣服を抱え、駆け出した。
```

### B. 土下座号泣

```
シャツが剥がれ、ベルトが緩み、ズボンが膝まで下がる。
サトシはその場で勢いよく土下座した。
サトシ:
  ご、ごめんなさいごめんなさいごめんなさいぃぃ！
  助けてください、許してくださいぃぃ！
涙と鼻水でぐしゃぐしゃの顔を床に擦りつけて、
サトシは何度も何度も詫びた。
ピー助:（呆れた声で）
  ...サトシ、それで許してもらえると思ってんのか？
  立て、行くぞ。
```

### C. 裸でフリーズ → ピー助救助

```
気が付くと、ほとんどの衣服が引き剥がされていた。
サトシは両手で前を押さえ、その場にしゃがみ込んで動けなくなる。
ピー助:（焦った声で）
  おい、サトシ！ いつまで固まってんだ！
  ...しょうがねえ、これで隠せ！
ピー助は近くの布切れを必死にたぐり寄せ、サトシの腰に巻きつけた。
サトシはピー助に引きずられるようにして、よろよろと退散した。
```

### D. 命乞い → ピー助の小突き

```
サトシ:（涙声で叫ぶ）
  な、なんでもしますぅ、許してくださいぃぃ！
  俺の命は、俺の命だけは、勘弁してくださいぃぃ！
脱がされかけた服を必死に押さえ、その場で泣き崩れるサトシ。
ピー助:（ため息混じりに）
  誰も命までは取らねえよ。...いいから走れ、サトシ。
ピー助に何度も小突かれ、ようやく動き出したサトシは、
転びそうになりながら、なりふり構わず駆け出した。
```

### E. 母を呼ぶ号泣 → ピー助に引きずられる

```
シャツの胸元が大きく裂け、ベルトが床に落ちる。
ついにサトシは、子供のように声を上げて泣き出した。
サトシ:
  うわぁぁぁんっ、お母さぁぁぁん！
ピー助:（深いため息）
  ...いい歳して何言ってんだお前は。
ピー助はサトシの腕を強引に引っ張り、半裸で号泣する主人を、
布の切れ端で前を隠したまま、引きずるようにして退散させた。
```

## 公開・中継系の例外条項

サブイベント 3 のように「真言の水晶球＋伝声塔」など、サトシの姿が
公衆に中継されうる演出が組み込まれている場合は、サトシ敗北時は
**必ず中継が停止／切断／失敗する** 設定にする。サトシの姿は世間に
晒されない。これは本ドキュメントのナレーション・パターンに優先する
ハードルールとして守る。
