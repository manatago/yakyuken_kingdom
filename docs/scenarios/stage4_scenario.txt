登場人物:
  サトシ       … 主人公。ベイズ・アイ持ち。王都での連勝が噂になり始めている
  ピー助       … 透明の聖霊ガイド。データ解析・電子錠ハック担当
  受付嬢       … 冒険者ギルドの受付。基本サトシに冷たいが、情報は正確に伝える
  セレス       … 魔法師団長。戦時参謀。国外の武力侵攻を魔法で退ける最終兵器
                 表：冷徹で理知的。部下に厳格。公の場での演説歴ゼロ
                 裏：ドM・自己緊縛マニア。自室に魔道ロープ多数。
                     「魔法縛の術式解析」という体裁の論文を大量執筆
                 動機：他の高官が次々とサトシに敗れる中、自分が次のターゲットに
                      なる前に先手を打つ（自己防衛＋研究欲＋暴かれたい倒錯の三層）
  魔法師団員   … セレスの副官・各種スタッフ。サトシを警戒している

シーケンス構成:
  stage4_pre       … 場面1（挑戦状）
  stage4_infiltrate … 場面2〜2.6（ピー助の潜入提案→警報発動→バレる→撤退）
  stage4_meet      … 場面3（対面・潜入疑いを突きつけ、秘密保持契約を賭け金に）
  stage4_battle1   … 場面4（縛鎖魔法で完敗）
  stage4_contract  … 場面4.5〜4.6（契約の執行→ピー助が契約解除＋性癖解析）
  stage4_minigame  … 場面5〜6（ミニゲーム準備＋縛鎖の結び目ほどき）
  stage4_battle2   … 場面7（再戦・勝利）
  stage4_post      … 場面8（契約は有効と信じるセレスの倒錯的執着）
  stage4_close     … 場面9（ギルド帰還）
