# キャラクターリグ作成ワークフロー

主人公（char01）の2Dボーンアニメーション作成手順書

## 概要

静止画の立ち絵に Skeleton2D / Bone2D を組み合わせて、呼吸・腕振り・頭の傾きなどの微細アニメーションを実現する。

## 素材

- **元画像**: `godot/assets/characters/samples/char01-skeleton-transparent.png`
- **パーツ構成**:
  - 右腕（左端）
  - 頭部（上部中央）
  - 体/胴体（下部中央）
  - 左腕（右端）

### Phase 2: ボーン構造作成

| # | 作業 | 担当 | 成果物 |
|---|------|------|--------|
| 2.1 | シーン雛形生成（Skeleton2D + Bone2D） | Claude | `char01_rig.tscn` |
| 2.2 | Godotでシーンを開く | 手動 | - |
| 2.3 | 各パーツ画像をSprite2Dとして仮配置 | 手動 | 位置確認用 |
| 2.4 | ボーン位置を画像の関節に合わせて調整 | 手動 | - |
| 2.5 | Rest Pose を設定 | 手動 | Skeleton2D → Set Rest Pose |

**ボーン階層**:
```
Skeleton2D
└── bone_spine (胴体の中心)
    ├── bone_head (首の付け根 → 頭頂)
    ├── bone_arm_r (右肩 → 右手)
    │   └── bone_hand_r (右手首 → 指先) ※オプション
    └── bone_arm_l (左肩 → 左手)
        └── bone_hand_l (左手首 → 指先) ※オプション
```

---

### Phase 3: ポリゴン作成・スキニング

| # | 作業 | 担当 | 成果物 |
|---|------|------|--------|
| 3.1 | 各パーツ用の Polygon2D ノード作成 | 手動 | 4つのPolygon2D |
| 3.2 | Polygon2D に Skeleton2D を割り当て | 手動 | skeleton プロパティ設定 |
| 3.3 | UV Editor で頂点を画像輪郭に配置 | 手動 | ポリゴンメッシュ完成 |
| 3.4 | Bones タブで頂点ウェイト設定 | 手動 | スキニング完成 |

**Polygon2D 設定手順（各パーツ）**:
1. Polygon2D ノード追加
2. `Texture` に該当パーツ画像を設定
3. `Skeleton` に Skeleton2D ノードを設定
4. `Polygon` メニュー → 「UV」→「Edit UV」
5. 頂点を画像の輪郭に沿って配置
6. 「Bones」タブで各頂点のウェイトをペイント
