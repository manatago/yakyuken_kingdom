# キャラクターリグ作成ワークフロー

主人公（char01）の2Dボーンアニメーション作成手順書

## 概要

静止画の立ち絵に Skeleton2D / Bone2D を組み合わせて、呼吸・腕振り・頭の傾きなどの微細アニメーションを実現する。

## 素材

- **元画像**: `godot/assets/characters/samples/char01-skeleton-transparent.png`
- **パーツ構成**:
  - 右腕（左端）
  - 頭部（上部中央）
  - 体/胴体（下部中央）
  - 左腕（右端）

#### 3.1 Polygon2D ノード作成

1. 「シーン」タブで **Sprites** を右クリック
2. 「子ノードを追加」を選択
3. **Polygon2D** を検索して選択
4. ノード名を変更（例: `PolygonBody`, `PolygonHead`）

---

#### 3.2 基本プロパティ設定（インスペクタ）

対応するSprite2Dの値を参照して設定する：

| プロパティ | 設定方法 |
|-----------|----------|
| **Texture** | 該当パーツ画像をドラッグ&ドロップ |
| **Skeleton** | `../../Skeleton2D` を入力（または NodePath で選択） |
| **Position** | 対応するSprite2Dと同じ値をコピー |
| **Z Index** | 対応するSprite2Dと同じ値をコピー |

**パーツ別設定値**:

| パーツ | Texture | Position | Z Index | 対応ボーン |
|--------|---------|----------|---------|-----------|
| Body | char01_body.png | (97.5, -359) | 0 | BoneSpine |
| Head | char01_head.png | (69.5, -1296) | 2 | BoneHead |
| ArmR | char01_arm_r.png | (-335.5, -278) | -1 | BoneArmR |
| ArmL | char01_arm_l.png | (442.5, -300) | 1 | BoneArmL |

---

#### 3.3 ポリゴン頂点の配置

1. Polygon2D ノードを選択
2. 下部パネルの **「ポリゴン」** タブをクリック
3. 画像の輪郭に沿ってクリックして頂点を配置
   - 関節部分は頂点を多めに配置（変形時に自然になる）
   - 直線部分は頂点を少なめでOK
4. 最後の頂点は最初の頂点の近くでクリックして閉じる

**ポイント**:
- 頂点数の目安: 15〜25点程度
- 時計回りまたは反時計回りで一貫して配置

---

#### 3.4 UV設定

1. 下部パネルの **「UV」** タブをクリック
2. ポリゴン頂点がテクスチャ上で正しい位置にあるか確認
3. ずれている場合は頂点をドラッグして調整

**補足**: ポリゴンタブで輪郭に沿って頂点を配置していれば、通常UVは自動的に正しくマッピングされる。

---

#### 3.5 ボーンウェイト設定

1. Polygon2D ノードを選択
2. 下部パネルの **「ボーン」** タブをクリック

```
下部パネル
┌─────┬──────────┬─────┬────────┐
│ 点  │ ポリゴン │ UV  │ ボーン │ ← これをクリック
└─────┴──────────┴─────┴────────┘
```

3. 右側パネルの **「ボーンをポリゴンに同期させる」** ボタンをクリック
   - ボーンリストが空白パネルに表示される

4. ボーンリストから**対応するボーン**を選択
   - Body → BoneSpine
   - Head → BoneHead
   - ArmR → BoneArmR
   - ArmL → BoneArmL

5. 上部ツールバーの **「ペイント」アイコン** をクリック（ペイントモード有効化）

6. **スライダーを最大値（1.0）** に設定

7. 左側の画像上で**全ての頂点をクリックまたはドラッグ**して塗る
   - **白** = ウェイトあり（正常）
   - **黒** = ウェイトなし

8. 全頂点が白くなったら完了

---

#### 3.6 動作確認

1. 「シーン」タブで `Skeleton2D` → 対応するボーン（例: `BoneSpine`）を選択

2. 「インスペクタ」の `Transform` → `Rotation` を変更（例: `10` や `-10`）

3. 確認ポイント:
   - ✅ **成功**: Polygon2D がボーンと一緒に回転する
   - ❌ **失敗**: Polygon2D が動かない → ウェイト設定を確認

4. 確認後、**Rotation を `0` に戻す**

---

#### 3.7 仮配置Sprite2Dの非表示化

ウェイト設定が完了したら、対応するSprite2Dを非表示にする：

1. 「シーン」タブで対応する Sprite2D（例: `SpriteBody`）を選択
2. インスペクタで **`Visible`** のチェックを外す
   - または「シーン」タブの**目のアイコン**をクリック

**補足**: 全パーツ完了後、Sprite2D は削除可能。作業中は非表示にしておくと戻しやすい。

---

#### Sprite2D と Polygon2D の違い

| 項目 | Sprite2D | Polygon2D |
|------|----------|-----------|
| 用途 | 画像をそのまま表示 | 画像を変形して表示 |
| ボーン追従 | ❌ できない | ✅ できる |
| 変形 | 拡大縮小・回転のみ | 頂点ごとに変形可能 |
| 作業での役割 | ボーン位置調整用の「ガイド」 | 本番用（最終的に使う） |

### 腕振りコントローラー (ArmSwingController) API

PrologueChapter.gd などのストーリースクリプトから呼び出せる腕振りアニメーションAPI。

#### 基本的な使い方

```gdscript
# CharacterRig インスタンスを取得
var rig: CharacterRig = $CharacterRig  # または適切なパス

# 腕振りを開始
rig.start_arm_swing({
    "angle_min": -10,      # 最小角度（度）
    "angle_max": 10,       # 最大角度（度）
    "duration": 2.0,       # 1サイクルの時間（秒）
    "loop": true,          # ループするか
    "repeat_count": 0,     # 繰り返し回数（0=無限）
    "mode": "alternate",   # モード
    "phase_offset": 180,   # 位相オフセット（度）
    "easing": "ease_in_out" # イージング
})

# 腕振りを停止
rig.stop_arm_swing()

# 腕をリセット
rig.reset_arms()
```

#### パラメータ詳細

| パラメータ | 型 | デフォルト | 説明 |
|-----------|------|---------|------|
| `angle_min` | float | -10 | 振り始めの角度（度） |
| `angle_max` | float | 10 | 振り終わりの角度（度） |
| `duration` | float | 1.0 | 1サイクルの所要時間（秒） |
| `loop` | bool | true | ループするかどうか |
| `repeat_count` | int | 0 | 繰り返し回数（loop=true時、0=無限） |
| `mode` | String | "alternate" | 動作モード（下記参照） |
| `phase_offset` | float | 180 | 左右の位相差（度、alternate/mirrorモード用） |
| `easing` | String | "ease_in_out" | イージング関数（下記参照） |

#### 動作モード (`mode`)

| モード | 説明 |
|--------|------|
| `alternate` | 左右の腕が交互に振れる（デフォルト） |
| `sync` | 両腕が同じ方向に同時に振れる |
| `mirror` | 両腕が逆方向に同時に振れる |
| `left_only` | 左腕のみ振る |
| `right_only` | 右腕のみ振る |

#### イージング (`easing`)

| 値 | 説明 |
|----|------|
| `linear` | 一定速度 |
| `ease_in` | ゆっくり始まる |
| `ease_out` | ゆっくり終わる |
| `ease_in_out` | ゆっくり始まり、ゆっくり終わる（デフォルト） |
| `bounce` | バウンス効果 |
| `elastic` | 弾性効果 |

#### シグナル

```gdscript
# 腕振り完了時（loop=false または repeat_count 到達時）
rig.get_arm_swing_completed_signal().connect(func():
    print("腕振り完了")
)

# 各ループ完了時
rig.get_arm_swing_loop_signal().connect(func(loop_index):
    print("ループ %d 完了" % loop_index)
)
```

#### 直接ボーン操作

```gdscript
# 左腕を直接回転（度）
rig.rotate_arm_left(15)

# 右腕を直接回転（度）
rig.rotate_arm_right(-15)

# 両腕を同時に回転
rig.rotate_arms(15, -15)
```
