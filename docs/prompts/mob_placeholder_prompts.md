# モブキャラ仮画像プロンプト集

未実装シナリオ（Stage2〜6 / Subevent3 / Subevent4）用のモブキャラについて、
**1キャラ1枚の仮画像（プレースホルダ）** を生成するためのプロンプト集。

---

## 共通事項

- **解像度**: 1024×1280（膝上構図、4:5縦長）
- **構図**: 膝上から頭頂まで映る構図、キャラクター中央配置
- **背景**: 単色白（背景削除スクリプトで透過化する前提）
- **作風**: アニメ調・セルシェード、線画くっきり、彩度やや抑えめ
- **含めないもの**: 背景、文字、エフェクト、不要な小物

## 生成後の処理フロー

各画像を生成したら、ダウンロードフォルダに保存して以下の手順を依頼:

1. **バックアップ**: `godot/assets/characters/<キャラ名>_backup/` にオリジナルを保存
2. **背景透過処理**: `python3 scripts/remove_bg_birefnet.py --erode 3 --smooth 1 <ファイル>`
3. **リサイズ**: 1024×2048 に統一（PIL/LANCZOS）
4. **配置**: 上記「配置先」の表に従って `<キャラ名>_default_001.png` として保存
5. **Godot import**: `--headless --import` で `.import` ファイル生成

依頼例:
> ダウンロード/sebas.png を仮画像として、mob/sebas/default/ に sebas_default_001.png として配置してください。背景透過とバックアップは忘れずに。
