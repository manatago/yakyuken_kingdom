# 未実装シナリオの新規背景プロンプト集

Stage2〜7 / Subevent3 で必要な新規背景17枚分のプロンプト。
各プロンプトは独立して動作するよう、意図・作風・除外事項をすべて含めています。そのままコピペ可。

## 共通事項

- **解像度**: 1344×768（16:9）
- **配置先**: `godot/assets/backgrounds/<stage>/<filename>`
- **作風**: 中世ファンタジーRPG、アニメ調・セルシェード、ビジュアルノベル風の背景画、線画くっきり、彩度やや抑えめ
- **構図**: 画面中央〜下1/3にキャラクター立ち絵を重ねる前提。その位置に密集させず余白を確保
- **除外**: 人物、キャラクター、動物、文字・ロゴ、吹き出し、UI要素
