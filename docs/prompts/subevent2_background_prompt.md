# サブイベント2 背景画像生成プロンプト（nano-banana用）

聖アレクシア教会関連の背景6枚。各プロンプトは独立して動作するよう、意図・作風・除外事項をすべて含めてあります。そのままコピペ可。

## 共通事項

- **解像度**: 1344×768（16:9）
- **配置先**: `godot/assets/backgrounds/subevent2/`
- **参考（雰囲気）**: 中世ファンタジーRPG、アニメ調、ビジュアルノベル風の背景画

---

## 1. 教会外観（bg_church_exterior.png）

```
中世ファンタジーRPGの2D背景画像を生成してください。

【描く対象】荘厳な石造りの聖堂の正面外観。ビジュアルノベル用の引きの背景。

【作風】アニメ調・セルシェード、線画くっきり、彩度やや抑えめ、暖色と冷色のコントラスト。解像度1344×768、アスペクト比16:9の横長ワイド。

【シーン詳細】
- ゴシック様式の聖堂、石造り。高い尖塔と鐘楼が左右対称に配置される
- 建物中央に大きな丸いステンドグラス（バラ窓）、青・赤・黄の色ガラス
- その下にアーチ型の木製の正面扉（閉じている）
- 前庭には石畳の広場、左右に古い街灯、正面扉へ続く石段
- 両脇に低い石塀、背後に木立
- 時刻は昼下がり、空はややグレーがかった曇り空
- 晴れてはいるが、人気のない冷たい雰囲気
- 光源は天空光、建物に柔らかい影

【含めないもの】人物、キャラクター、動物、文字・ロゴ、吹き出し、UI要素

【構図の注意】画面中央〜下1/3にキャラクター立ち絵を重ねる前提のため、その位置には建物や小物を密集させず余白を確保する。
```

【描く対象】教会の隠し部屋「特別礼拝室」。表向きは高額献金者の祈祷室だが、実態は覗き部屋。悪趣味なVIP空間。バトルシーンの背景としても使用。

【作風】アニメ調・セルシェード、線画くっきり、彩度やや抑えめ、暖色と冷色のコントラスト。解像度1344×768、アスペクト比16:9の横長ワイド。

【シーン詳細】
- 小さく豪奢な室内空間。薄暗くロウソクの明かり
- 画面右側の壁に3〜4個の小さな覗き穴が縦に並ぶ（直径5cm程度、不規則な間隔、かすかな光が漏れる）
- 部屋の奥側に豪華な革張りのひとり掛け椅子（貴族風、金の装飾）

- 床は暗色の絨毯
- 光源はロウソクと覗き穴から漏れる光のみ、全体的に薄暗い

【構図の注意】バトル背景として使うため、画面中央に1人のキャラクター立ち絵を重ねる余白を確保（椅子・机・帳簿は左右に寄せる）。覗き穴は右寄りに配置。

## ファイル配置先

シーン順に並ぶよう番号プレフィックスを付ける：

```
godot/assets/backgrounds/subevent2/
├── bg01_church_exterior.png    # 場面2 教会外観
├── bg02_church_interior.png    # 場面3 礼拝堂内部
├── bg03_church_backyard.png    # 場面3 裏庭
├── bg04_church_corridor.png    # 場面4 廊下
├── bg05_church_peep_room.png   # 場面4 特別礼拝室（バトル背景兼用）
└── bg06_church_dungeon.png     # 場面4.5 地下牢（敗北時）
```

## 生成後の実装切り替え

`godot/story/chapters/Subevent2Chapter.gd` 冒頭の定数を更新：

```gdscript
const BG_CHURCH_EXT := "res://assets/backgrounds/subevent2/bg01_church_exterior.png"
const BG_CHURCH_INT := "res://assets/backgrounds/subevent2/bg02_church_interior.png"
const BG_CHURCH_BACKYARD := "res://assets/backgrounds/subevent2/bg03_church_backyard.png"
const BG_CHURCH_CORRIDOR := "res://assets/backgrounds/subevent2/bg04_church_corridor.png"
const BG_PEEP_ROOM := "res://assets/backgrounds/subevent2/bg05_church_peep_room.png"
const BG_DUNGEON := "res://assets/backgrounds/subevent2/bg06_church_dungeon.png"
```

シスター長バトルの背景（`SisterBattleChapter.gd`）：

```gdscript
func get_battle_background() -> String:
    return "res://assets/backgrounds/subevent2/bg05_church_peep_room.png"
```
