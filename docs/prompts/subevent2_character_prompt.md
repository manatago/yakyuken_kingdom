# 運用メモ

- **段階生成の推奨順序**:
  1. 各キャラの「基本形」（サトシ-1、受付嬢-1、番兵-1、シスターA-2、シスター長A-1）をまず生成してキャラデザを確立
  2. 基本形を参考画像に、残りの表情バリエーションを安定生成
  3. シスター長の【B】【C】【D】は各ベースから順に生成
- **過度な露出回避**: バトル中outfit1や敗北後の乱れは、下着のちらりや肩・鎖骨程度に留める
- **背景削除**: 生成後、`scripts/remove_bg_birefnet.py` で背景を透過化
- **配置**: 生成後は `godot/assets/characters/subevent2/` にキャラ別の命名で配置
