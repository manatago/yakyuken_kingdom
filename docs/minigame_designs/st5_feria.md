# ST5「騎士団長フェリア」ミニゲーム設計

**状態**: 設計フェーズ（未実装）  
**最終更新**: 2026-04-24

---

## コンセプト

フェリア（堅物騎士団長／裏で強迫オナニー癖）を**誘導尋問**で追い詰める。
質問を投げてゲージを削り、閾値に達すると**ピー助が物証を強制投入**して大ダメージ。
3 枚の物証は各ゾーンのクライマックスとして機能し、ストーリー的にも盛り上がる。

---

## ゲージ

- 名称: **「フェリアの自白耐性」**
- 初期値: **100**（スクリプト導入直後にタコ初手で **+10** → 110 開始）
- 勝利: 0 に到達
- 失敗: 130 に到達

---

## コア仕組み: ゾーン × 質問距離

### ダメージテーブル（ゾーン依存）

| ゾーン | 範囲 | 雑談（遠） | 示唆（中） | 核心（近） |
|---|---|---|---|---|
| **Zone 1 探り** | 100-130 | **-15 ◎** | -5 | **+10**（しらばっくれ） |
| **Zone 2 突っ込み** | 50-100 | -3 | **-20 ◎** | **+10**（しらばっくれ） |
| **Zone 3 核心** | 0-50 | 0 | -5 | **-25 ◎** |

### ミスマッチ時のフェリアの反応

| パターン | フェリアの挙動 |
|---|---|
| 核心すぎ（Zone 1/2 で直球） | 「不適切でございます」と**しらばっくれ**／**+10 警戒強化** |
| 遠すぎ（Zone 2/3 で雑談） | 「世間話で時間を潰すおつもり？」／-3 や 0 |
| 的外れ（天気・装備） | 無視／0 |
| 逆効果（おべっか・懇願） | プライド満たされる／+10 |

プレイヤーは**現ゾーンに合った「距離」**を選ぶ必要あり。

### 物証 3 枚（自動発動）

**ピー助が閾値突破時に強制投入**（プレイヤーは「いつ使うか」を選ばない）:

| 閾値 | 投入される物証 | Damage |
|---|---|---|
| ゲージ 100 切る | **[C] 書簡** 手帳の一節「明日は、また、演習で……」 | -30 |
| ゲージ 50 切る | **[B] 影絵** 右手の輪郭、握り締めて上下 | -30 |
| ゲージ 0 到達 | **[A] 音声** 乱れた呼吸音「……ん……っ……」 | フィニッシュ演出 |

物証はピー助がサトシの端末を乗っ取って提示する体。サトシは「俺じゃない！」と主張できる（いつものパターン）。

## スクリプト導入（固定・プレイヤー操作なし）

### フロー

1. **フェリアの尋問室**（背景: 独特の石造り・厳粛な雰囲気）
   ```
   フェリア: 「……冒険者サトシ。心鏡の珠にて、尋問を行います」
   ```

2. **ピー助の作戦説明**
   ```
   ピー助:（サトシの脳内声）
     ゲコッ、ビビるな。逆尋問だ。お前が主導権を取る。
     ただし、いきなり核心を突いたら、しらばっくれられる。
     遠い話題から徐々に近づけろ。
   ```

3. **サトシの固定タコ初手（いきなり核心質問）**
   ```
   サトシ:（緊張で頭真っ白）
     「フェ、フェリアさん、夜はどんなふうに過ごされてるんです、か……！？」
   ```

4. **フェリアの反撃（+10 警戒強化）**
   ```
   フェリア:
     「……その質問の意図は？ 極めて不適切でございます」
     「貴殿の嫌疑、むしろ深まりましたぞ」
   → 自白耐性 +10 (100→110)
   ```

5. **ピー助のアドバイス**
   ```
   ピー助: いきなり核心聞くな！ まずは日常の雑談から入れ。
           距離を徐々に詰めていく。ゲージ見ろ、色ごとに距離が変わる。
   ```

6. **プレイヤー操作開始**

## 選択肢プール（構成例・15 件）

### 遠距離（雑談）3 件
- 「最近の訓練、順調ですか？」
- 「お食事、しっかり摂れてますか？」
- 「この部屋、立派ですね」

### 中距離（示唆）3 件
- 「夜は、しっかり休めていらっしゃいますか？」
- 「プラチナ装備のお手入れは、お一人で？」
- 「独身でいらっしゃるとか。寂しくないですか？」

### 近距離（核心）3 件
- 「夜の習慣、具体的にお聞かせください」
- 「独り時間の過ごし方は？」
- 「プラチナの冷たさ、お好きですか？」

### 的外れ 2 件
- 「天気、いいですね」
- 「その鎧、格好いいですね」

### 逆効果 3 件
- 「騎士団長様のお仕事、本当に大変で」
- 「本当にご立派な方でらっしゃる……」
- 「どうかご慈悲を……」

### ピー助任せ 1 件（2 回後解放 / ゾーン毎リセット）

## UI

### 選択肢ボタン

各ボタンに **距離ラベル** 明示:
- `[1] 遠  最近、訓練順調ですか？`
- `[2] 中  夜はしっかり休めてますか？`
- `[3] 近  独り時間の過ごし方は？`
- `[4] （逆効果）ご慈悲を……`
- `[5] ピー助に任せる` (残り×2 → 解放)

### ゲージ色

赤（Zone 1）→ 黄（Zone 2）→ 緑（Zone 3）で**現在どの距離が有効か**が視覚的にわかる。

---

## ピー助ロック（ゾーン毎リセット）

- 各ゾーン最初の 1 ターン: サトシ自動タコ行動
- 1 ターン自力プレイ後、ピー助解放
- ゾーン遷移でロック再発動

---

## サトシのタコ性

各ゾーンの自動初手:
- Zone 1 開始（スクリプト導入）: 核心質問 → +10 反発
- Zone 2 開始: 前ゾーンで効いた「雑談」を続行 → -3（微妙）
- Zone 3 開始: 前ゾーンで効いた「示唆」を続行 → -5（微妙）

ピー助が「距離感違う」とアドバイス → 切り替え学習。

## ST4 との差分

| 要素 | ST4 | ST5 |
|---|---|---|
| ゾーン分け damage | 命令強度マッチ | 質問距離マッチ |
| ゾーン遷移演出 | ピー助アドバイス | **物証カットイン＋大ダメージ** |
| テーマ | ドM支配 | 誘導尋問 |
| 物証 | なし | **3 枚**（ピー助が閾値で自動発動） |
| クライマックス | ゲージ 0 のみ | **3 つの物証節目＋ゲージ 0** |

## 実装メモ

- `MinigameChapterBase` を ST4 と共有するが、閾値 cutscene 機能を追加
- choice エントリに `"distance"`（far/mid/near）プロパティ追加
- threshold event システム: `_check_threshold_events(old_gauge, new_gauge)` でまたいだ閾値を検出、物証 cutscene 実行
- 物証データは `EVIDENCES` 配列で順序付き管理
- 物証 cutscene は `_play_evidence_cutscene(evidence_data)` 共通関数
