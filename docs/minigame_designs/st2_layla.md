# ST2「アサシン・レイラ」ミニゲーム設計（案）

**状態**: 設計フェーズ（未実装）  
**最終更新**: 2026-04-24

---

## コンセプト

レイラ（超純情アサシン）の動揺ゲージを 110 → 0 まで削る。
プレイヤーは 3 カテゴリから 1 つずつ選び「言葉責め弾」を構築する。
**各ラウンドのシーン画像に特徴**があり、それぞれに正解セットがある。観察して推理するパズル要素。

---

## ゲージ

- 名称: **「レイラの冷静度」**
- 初期値: **110**
- 勝利: 0 に到達
- 失敗: 130 に到達

---

## 選択肢（3スロット × 3オプション）

| スロット | 選択肢 |
|---|---|
| 対象 | 手 / 唇 / 胸元 |
| 状態 | 震えてる / 紅潮してる / 汗ばんでる |
| 断定 | 初めて / 訓練だけ / 実戦ゼロ |

---

## スコア計算

各ラウンドにはシーン画像＋正解セット `(対象, 状態, 断定)` が存在する。

| 一致数 | ダメージ |
|---|---|
| 3 つ一致（完全正解） | **-50**（クリティカル） |
| 2 つ一致 | -30 |
| 1 つ一致 | -15 |
| 0 つ一致 | **+5**（シラけて逆効果） |
| ピー助任せ | **-60**（自動正解＋余計な一言ボーナス） |

---

## ラウンド構成（案）

- 毎ターン、レイラの異なるシーン画像＋短い描写が表示される
- プレイヤーは画像・描写から正解を推理して選択
- シーン画像プール: 5〜7 パターン作成してランダム選出

### シーン例

| シーン描写 | 画像の特徴 | 正解 |
|---|---|---|
| 「刀を振り上げた瞬間、手元が微かに震えている」 | 振り上げポーズ、手に注目 | 手 / 震えてる / 初めて |
| 「唇を噛みしめ、目元が赤く染まる」 | 表情アップ、唇に注目 | 唇 / 紅潮してる / 訓練だけ |
| 「襟元が汗で湿って張り付いている」 | 胸元に注目、汗で光る | 胸元 / 汗ばんでる / 実戦ゼロ |

---

## ピー助ボタン

- **2 ラウンド以上** プレイヤーが自力で挑戦してから解放（※全ミニゲーム共通ルール）
- 解放前はボタン非アクティブ＋説明ツールチップ「サトシが 2 回試した後に使える」
- 使うと自動で正解 3 セットを選択＋余計な煽り台詞ボーナスで **-60**

---

## 画像素材

**要作成**: レイラの各シーン画像（5〜7 枚）  
現状は `layla_001.png`（全身立ち絵）のみ。ミニゲーム実装前に追加画像が必要。

**仮代替案**: 画像は共通の 1 枚を使い、**シーン描写テキスト**で差別化して実装 → 画像が揃ってから差替。
